# Security Policy

This repository is for **lab work only**. Do not submit real vulnerabilities from third-party systems here.

## Reporting Issues
If you find a security issue *with the content of this repo* (e.g., leaked credentials in a sample), open a private issue or contact the maintainer directly. Do not include secrets in public issues.

## Scope
- Lab configs, detection rules, and educational scripts.
- No active testing on assets you do not own or lack explicit permission to assess.
