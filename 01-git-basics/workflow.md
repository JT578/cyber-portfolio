# Git: Tiny Workflow

## Clone via SSH
```
git clone git@github.com:<user>/<repo>.git
```

## Create a branch, commit, push
```
git checkout -b week01-setup
git add .
git commit -m "week01: initial setup and notes"
git push -u origin week01-setup
```
Open a Pull Request if you're practicing PR flow.

## Update local
```
git pull
git fetch -p
```
