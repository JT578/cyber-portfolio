# Cybersecurity Career Paths — Week 1 Hub

**Status:** starter bundle generated on 2025-09-04  
**Goal:** Stand up your tooling, lock in Git/SSH, and curate your initial intel sources so you can ship work without yak‑shaving.

## Index
- 00-accounts-and-setup/
  - `ssh-setup-windows.md` — Ed25519 keypair, agent, GitHub SSH, verification
  - `git-setup.md` — identity, defaults, and first push (SSH)
  - `tools-to-install.md` — minimal, fast installs
- 01-git-basics/
  - `workflow.md` — branch/commit/push basics, cloning via SSH
- 02-linux-basics/
  - `nav-perms.md` — navigation, sudo, permissions, package updates
- 03-networking-basics/
  - `tcpip-dns.md` — IP, ports, TCP vs UDP, DNS lookups
- links-and-resources/
  - `people-and-repos.md` — people to follow, repos to star
  - `learning-platforms.md` — HTB, THM, OWASP references
- evidence/
  - Drop screenshots of successful steps here (SSH test, first push, etc.)

## Checklist
- [ ] Install core tools (terminal, editor, git, OpenSSH)
- [ ] Generate Ed25519 key and add it to GitHub
- [ ] Enable SSH agent and add key
- [ ] Configure git username/email and sane defaults
- [ ] Create or update your master repo for college
- [ ] First commit and push over SSH
- [ ] Star/key follow sources in `links-and-resources/people-and-repos.md`
- [ ] Basic Linux drills (cd/ls/cp/mv/grep/find/chmod/chown)
- [ ] Basic networking drills (ipconfig/ifconfig, ping, traceroute, nslookup/dig)

## Usage
Clone this bundle into your working repo or copy the `week01/` folder and commit it. Keep proofs in `evidence/` and link them from the markdown files as you go.
